So what do I do?

Alright, so if you have no idea with what to do, here's how you start. First thing you have to do is pull up your text editor; I'm using Notepad here. Start your document with <!DOCTYPE html>, which will tell the browser what version of HTML you are writing in. After that follow it with <html> and </html>

In html, these short combinations letters within the arrows are called tags. They generally come in pairs, defining a certain body of code within it. The first tag opens the container for the code, and the second one closes it. The second tag will have a backslash (/) in front of the letters to indicate it is closing. In this case, the tags <html> and </html> will contain code that will be interpreted as html (surprise). To start making the body of the page, add the tags <body> and </body> within the body of the html tags.

So far, we haven't actually made anything show up on the page. Let's start with the title of the page. The tag for headings in html are <h1> (and closed with <h1>), where you can change the number to make the font size smaller. Say you want to write the name of your pbro (say it's Jordan Shen). Simply write his name in between the tags. Put this title within your body tags.

Let's write something about our favorite pbro. This time, we'll use the tag <p> (closed with </p>, p stands for paragraph). Write whatever you want in between the lines.

Now, let's put in a picture of Jordan. This is a little different, because images don't need a closing tag; the <img> tag stands alone. Instead of putting information on the element in between two tags, we're going to put the information inside the brackets. Specifically, we're going to write the source url as an attribute in the angle brackets of the tag. Write src and set it equal to your image's url in quotes. You can find some picture on Facebook or Instagram, right-click it and copy the image URL to here

Last step, putting in the link to his facebook. The tag for links are <a> (and closed with </a>). We will another attribute (so placing something within the brackets of the first tag) to define where we are linking to. To do that, we set href equal to the link we want to go to (href="facebook.com/url"), all in the <a> tag. But now we need text to click. On the inside of the two tags, we'll write "click me" or something.
